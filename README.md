# QuantityMeasurementApp

**Overview**
QuantityMeasurementApp is a C# console application for handling length measurements. It supports equality checks, unit conversions, and arithmetic operations acrossmultiple units (feet). The project demonstrates OOP concepts, DRY principle, and scalable design.

Use Cases
UC1: Feet Measurement Equality

Compares two feet values for equality.
Handles null, non-numeric, and type validation.
Tests: same value, different value, null, same reference.
